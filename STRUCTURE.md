# STRUCTURE — NOCTURNE (bar-sample)

단일 파일 정적 사이트. `index.html`에 HTML/CSS/JS 인라인.

## 무드/축 (매트릭스)
- 무드(F): **네온 나이트라이프 / 미드나잇+딥주얼+네온 글로우** (기존 11종에 없던 생동감 다크. GOYO의 차분한 다크와도 구별)
- 페이지모델(A): 단일 롱스크롤, 풀블리드 다크 섹션
- 주인공(B): 시그니처 칵테일 + 공간 무드
- 레이아웃(C): 중앙 드라마틱 + 메뉴 리스트
- 내비(D): 상단 미니멀 + 실시간 영업상태 배지
- 시그니처(E): **실시간 영업 상태(시간 기반 영업중/곧 마감/마감)** + 네온 글로우 호버 — 신규 인터랙션

## 섹션 구성
1. 상단바(fixed) — 워드마크·내비·영업상태 배지·예약 CTA
2. 히어로 — 네온 워드마크·태그라인·영업상태·CTA(예약 문의/메뉴)
3. 시그니처 칵테일 — 이름·설명·ABV·가격(정직: 메뉴는 예시)
4. 공간 갤러리 — 무디 공간 컷
5. 운영시간 / 라스트오더 — 요일별·휴무 (바 필수 정보)
6. 예약·문의 폼 — 공통 AJAX 핸들러(데모 성공/실연결 2xx)
7. 찾아오기 — 주소·지도(다크 폴백)·네이버 지도
8. 푸터 — ©2026·SNS
9. 모바일 하단바 — 전화 · 예약 문의 (PC 메인 CTA와 일치)

## 적용 세이프가드 (LESSONS)
- reveal + rescue(숨김/백그라운드 탭 대비), 페이드 0.45s
- 공통 폼 핸들러(인페이지 성공/실패·중복제출 방지·데모 가짜성공/실ID 2xx)
- scroll-padding-top, OG/트위터 태그 + SVG 파비콘, ©2026
- 모바일 하단바 채널 = PC 메인 CTA 채널 일치
- 지도: OSM iframe + 다크 배경 폴백(타일 실패 시 흰 박스 방지)
- word-break:keep-all, 고정바=div, 100svh, 스크롤 핸들러 passive
- 이미지: Unsplash, 사용 전 로드 검증 + 외국어 간판 배제
